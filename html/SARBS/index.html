What SARBS is

SARBS (Soyless Auto Rice Bootstrapping Script) is a shellscript for installing a collection of soyless programs and configuration files on Arch-based systems.

What SARBS is not

Soydev friendly.

Usage

Run 'curl -LO https://soyless.xyz/sarbs.sh' and execute the script with root privileges. Nothing bad will happen. I promise.

Goal

Install a fully customized environment with minimal user intervention.
