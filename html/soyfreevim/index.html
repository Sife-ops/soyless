What's SoyFreeVim?

SoyFreeVim is a fully modular configuration file for (Neo)Vim. To enable a module, move it from modules/ to init.d/. Modules are loaded in numerical order.

Available modules

  • Conquer of Completion
  • Tmux Navigator
  • Tmux Resizer
  • Gruvbox
  • Undotree
  • Vim surround
  • Vim commentary
  • Vimwiki
  • ... and many more!
