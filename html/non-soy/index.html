What is non-soy?

Non-soy means soyless software, ie. based software. Examples of soyless software are:

  • Operating System
    • Linux
    • BSD

  • Window Manager
    • dwm
    • bspwm
    • xmonad
    • i3
    • xfce

  • Terminal
    • st
    • alacritty

  • Text Processor
    • Emacs
    • Vim

  • Document Authoring
    • Groff
    • LaTeX

  • Multimedia
    • mpd
    • mpv

  • Images
    • sxiv
    • Gimp

  • Remote Computing
    • SSH
    • rsync
